\section{Introduction}
This document describes the implementation of a 32-bit RISC-V core (RV32I) developed in VHDL.

\section{Architecture}
The core follows a classic RISC pipeline structure:
\begin{itemize}
    \item \textbf{Fetch:} Instruction retrieval from Memory.
    \item \textbf{Decode:} Opcode analysis and register file access.
    \item \textbf{Execute:} ALU operations and address calculation.
\end{itemize}
